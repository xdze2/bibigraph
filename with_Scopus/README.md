# Bibigraph
scientific citation graphs visualisation


- Get the data from [Scopus](https://www.scopus.com) (need an account), export to bibtex.




- http://beakerx.com/documentation

## install

    $ pipenv install
    $ pipenv shell
    $ pip install beakerx
    $ beakerx install
    $ jupyter nbextension enable beakerx --py --sys-prefix

    $ jupyter notebook
